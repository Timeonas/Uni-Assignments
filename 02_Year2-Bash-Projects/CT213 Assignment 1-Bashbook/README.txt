Download the Scripts directory provided.
Ensure the directory is on your desktop (~/Desktop/Scripts)
Navigate to the Scripts directory on the Desktop using the Linux terminal (cd Desktop/Scripts). 
Run the server.sh file (./server.sh) to create the Bashbook user directory
Run any of the availible commands with the necessary parameters.

Availible Commands and Parameters:
create (userid)
add (userid) (friendid)
post (senderid) (receiverid)
display (userid)

To exit the program, press control z.
All the necessary user information will be held within the Bashbook directory on your Desktop
